# 7. Recommended Tech Stack

| Layer           | Technology                                     |
| --------------- | ---------------------------------------------- |
| Frontend        | React.js (or Angular)                          |
| Backend         | Spring Boot                                    |
| Database        | MySQL                                          |
| ORM             | Spring Data JPA (Hibernate)                    |
| Authentication  | Spring Security + JWT (or session-based login) |
| Email           | JavaMailSender                                 |
| Build Tool      | Maven                                          |
| IDE             | IntelliJ IDEA / Eclipse                        |
| Version Control | Git                                            |

# 8. Project Workflow

```
Admin Login
      ↓
Manage Users & Departments
      ↓
Create Email Template
      ↓
Create Campaign
      ↓
Select Difficulty
      ↓
Set Redirect Count
      ↓
Send Phishing Email
      ↓
Employee Opens Email
      ↓
Employee Clicks Link
      ↓
Redirect Tracking
      ↓
Credential Submission
      ↓
Risk Score Calculation
      ↓
Department Ranking
      ↓
Dashboard & Reports
```

This structure is appropriate for a final-year project because it separates concerns into clear modules, uses a standard layered Spring Boot architecture, and naturally incorporates the additional features you've discussed, such as campaign difficulty, redirect tracking, suspicious keyword analysis, risk scoring, and department ranking.
